# Buscador de Productos en Tiempo Real

Práctica de JavaScript usando manipulación del DOM.  
Este proyecto consiste en un buscador de productos que filtra una lista en tiempo real conforme el usuario escribe en el input y muestra la cantidad de resultados encontrados.

## Descripción

El programa toma el texto ingresado por el usuario, revisa cada producto de la lista y muestra solo los que coinciden con la búsqueda.  
Además, actualiza automáticamente el contador de resultados visibles.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Lista de productos en pantalla
- Campo de búsqueda dinámico
- Filtrado en tiempo real
- Contador de resultados actualizado automáticamente
